О проекте

Father Code Token (FCD) — это утилити-токен бренда «Отцовский Ремень», работающего с 2010 года. Проект объединяет торговлю, музыку, вирусные клипы TikTok/Instagram, токен FCD и Web3-инфраструктуру.
